## About

Simple dashboard developed using NextJS (Framework), Prisma (ORM), MUI (UI Framework)

### Language

-   NodeJS

## Setup

-   To mirgrate

```sh
# for npm users
npx prisma migrate dev
# for pnpm users
pnpm prisma migrate dev

```

-   To generate a Prisma client

```sh
# for npm users
npx prisma generate
# for pnpm users
pnpm prisma generate

```

-   To browse database

```sh
# for npm users
npx prisma studio
# for pnpm users
pnpm prisma studio

```
